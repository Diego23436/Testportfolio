Project Title

Portfolio Website – Personal Web Portfolio

Project Description

This project is a simple personal portfolio website that showcases my skills, projects, and contact information. It is designed to present my work in a clean and professional way.

Technologies Used
	•	HTML – structure of the website
	•	CSS – styling and layout
	•	Git – version control
	•	GitHub – repository hosting and project management

How to View/Run the Project
	1.	Clone the repository:

git clone <repository-url>


	2.	Open the project folder.
	3.	Open index.html in any web browser to view the website.

Git Concepts Practiced
	•	Initialising a repository
	•	Adding and committing changes (git add, git commit)
	•	Pushing to GitHub (git push)
	•	Writing a proper README
	•	Basic file version tracking
